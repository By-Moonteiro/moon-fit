# Ideias e Features Futuras

> Este arquivo é um dump organizado de ideias.
> O que está aqui NÃO é compromisso — é direção.

---

## Fase 1 — Personal Tool (MVP e primeiras evoluções)

### MVP (escopo atual)

- [ ] Criar fichas de treino com exercícios pré-cadastrados
- [ ] Modo Treino ativo (cronômetro de descanso, marcar série, tempo total)
- [ ] Ciclo de vida da ficha (aviso de vencimento, sugestão de nova ficha)
- [ ] Histórico de fichas anteriores com anotações
- [ ] Anotações por exercício e por sessão
- [ ] Editar informações do usuário (nome, email, senha)

### Pós-MVP

- [ ] Gifs/videos de execução dos exercícios
- [ ] Login com Google
- [ ] PWA para melhor integração mobile
- [ ] Anotações sobre dieta e evolução de peso

---

## Fase 2 — Para Profissionais

- [ ] RBAC com níveis de acesso (Admin, Instrutor, Personal, Aluno)
- [ ] Gestão de alunos e fichas de terceiros
- [ ] Gerenciamento de mensalidades
  - [ ] Notificação de vencimento
  - [ ] Bloqueio de acesso para inadimplentes
- [ ] Formas de pagamento (Pix, QR Code)
- [ ] Geração de PDF por dia de treino
- [ ] Versão mobile nativa (React Native ou Flutter)

---

## Fase 3 — Inteligência / IA

- [ ] Geração de fichas personalizadas por IA
  - [ ] Baseada no perfil e histórico do usuário
  - [ ] Com possibilidade de edição após geração
- [ ] IA com acesso ao histórico de treinos e anotações
  - [ ] Para sugestões mais embasadas e personalizadas

## Dump livre (ideias soltas, sem compromisso)

- Sistema de Level baseado em horas treinadas
- Sistema de conquistas/badges por metas atingidas
- Comparativo visual de evolução (gráficos de carga, tempo, volume)
- Compartilhamento de fichas entre usuários
- Integração com wearables (frequência cardíaca, etc.)
